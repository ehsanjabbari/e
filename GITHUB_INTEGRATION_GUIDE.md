# راهنمای ادغام با GitHub

## مرحله ۱: ایجاد Personal Access Token

### 1.1 ورود به GitHub
1. به [GitHub.com](https://github.com) بروید
2. وارد حساب کاربری خود شوید

### 1.2 ایجاد Personal Access Token
1. روی عکس پروفایل خود در بالا کلیک کنید
2. **Settings** را انتخاب کنید
3. در منوی سمت چپ، **Developer settings** را انتخاب کنید
4. **Personal access tokens** → **Tokens (classic)** را انتخاب کنید
5. روی **Generate new token** کلیک کنید

### 1.3 تنظیم دسترسی‌ها
```
✅ gist          (دسترسی به GitHub Gists)
✅ repo          (برای private repositories - اختیاری)
```

### 1.4 کپی کردن توکن
- توکن را کپی کنید
- ⚠️ **مهم**: توکن فقط یک بار نمایش داده می‌شود!

---

## مرحله ۲: راه‌اندازی در اپلیکیشن

### 2.1 ورود توکن
1. در اپلیکیشن، به تب **تنظیمات** بروید
2. در بخش **📤 اشتراک‌گذاری GitHub**:
   - توکن کپی شده را در فیلد "توکن Personal Access" وارد کنید
   - روی **ذخیره تنظیمات** کلیک کنید

### 2.2 ایجاد اولین Gist
یکی از روش‌های زیر را انتخاب کنید:

#### روش A: ایجاد مستقیم از اپلیکیشن
1. در همان بخش، روی **ایجاد Gist جدید** کلیک کنید
2. منتظر بمانید تا Gist ایجاد شود

#### روش B: ایجاد از سایت GitHub
1. به [gist.github.com](https://gist.github.com) بروید
2. یک Gist جدید ایجاد کنید:
   - نام فایل: `inventory-backup.json`
   - محتوا: هر متنی (بعداً خودکار جایگزین می‌شود)
3. ID یا URL Gist را کپی کنید
4. در اپلیکیشن، ID را در فیلد "شناسه Gist" وارد کنید

---

## مرحله ۳: استفاده روزانه

### 3.1 بکاپ گرفتن
1. داده‌های خود را وارد کنید
2. در تب تنظیمات، روی **ارسال بکاپ به GitHub** کلیک کنید
3. منتظر پیام موفقیت بمانید

### 3.2 بازیابی داده‌ها
1. در تب تنظیمات، روی **بارگیری از GitHub** کلیک کنید
2. داده‌ها از Gist بارگیری می‌شود
3. اپلیکیشن به‌روزرسانی می‌شود

---

## ویژگی‌های امنیتی

### ✅ نقاط قوت
- **توکن در localStorage ذخیره می‌شود** (فقط در مرورگر شما)
- **Gist عمومی** (بدون نیاز به احراز هویت برای خواندن)
- **فقط شما دسترسی نوشتن دارید** (توکن شخصی شما)

### ⚠️ نکات امنیتی
- توکن خود را به اشتراک نگذارید
- توکن را در کد عمومی قرار ندهید
- در صورت مفقود شدن، توکن را در GitHub غیرفعال کنید

## رفع مشکلات

### مشکل: "خطا در احراز هویت"
```
راه حل: 
1. توکن را مجدداً کپی کنید
2. اطمینان حاصل کنید که توکن هنوز معتبر است
3. مجوزهای توکن را بررسی کنید
```

### مشکل: "Gist یافت نشد"
```
راه حل:
1. ID یا URL صحیح Gist را وارد کنید
2. اگر Gist عمومی نیست، باید توکن داشته باشید
```

### مشکل: "خطای CORS"
```
راه حل:
1. چند دقیقه صبر کنید
2. اتصال اینترنت را بررسی کنید
3. دوباره تلاش کنید
```

---

## جایگزینی محیط کاری

### انتقال به کامپیوتر جدید:
1. همه فایل‌های اپلیکیشن را کپی کنید
2. توکن GitHub را دوباره وارد کنید
3. Gist ID را وارد کنید
4. تست کنید که بکاپ/رستور کار می‌کند

### انتقال بین دستگاه‌های موبایل:
1. در دستگاه اول: **ارسال بکاپ به GitHub**
2. در دستگاه دوم: **بارگیری از GitHub**
3. همگام‌سازی کامل!

---

## پیشرفته: اتصال به GitHub Pages

### برای دسترسی آنلاین:
1. فایل‌های اپلیکیشن را به ریپازیتوری GitHub آپلود کنید
2. **Settings** → **Pages** → Enable GitHub Pages
3. آدرس سایت: `https://username.github.io/repository-name`
4. حالا همه جا به اپلیکیشن دسترسی دارید!
